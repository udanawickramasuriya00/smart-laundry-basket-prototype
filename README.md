# Smart Laundry Basket Prototype

This project is a prototype concept for a smart laundry basket designed to improve hygiene and laundry management using IoT technology.

## Technologies Used
- Arduino
- UV-C light module
- Bluetooth module
- Sensors

## Features
- UV-C light system for bacteria reduction
- Odor control system concept
- Compression mechanism to reduce laundry volume
- Bluetooth connectivity for mobile alerts

## Project Status
Prototype stage – developed within a short development period to demonstrate the concept of a smart laundry management system.
